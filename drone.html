Smart Agri-Drone IoT

Système complet d'agriculture de précision interconnecté. Fusionne les données aériennes (Drone Autonome) et terrestres (Sondes ESP32) pour une surveillance temps réel.

Raspberry Pi + Navio2 Matlab Simulation IoT Protocol Web Dashboard

Architecture Distribuée

La force de ce projet réside dans la communication entre trois entités distinctes :

  • Le Vecteur Aérien : Un drone quadcoptère autonome qui scanne la zone et mesure les conditions atmosphériques.
  • Le Réseau Terrestre : Des nœuds fixes basés sur ESP32 plantés dans le sol pour remonter l'humidité de la terre.
  • Le Hub de Contrôle : Une interface centrale qui agrège toutes les données via un protocole IoT pour aider l'agriculteur à décider.

1. Hardware & Navio2

Contrairement aux contrôleurs de vol "boîte noire", j'ai opté pour une approche Linux embarqué pour une flexibilité totale.

  • Cœur : Raspberry Pi 4 couplée à un HAT Emlid Navio2. Cela permet de faire tourner ArduPilot tout en exécutant des scripts Python en parallèle.
  • Capteurs : Le drone transporte ses propres capteurs de température/humidité pour cartographier le micro-climat.

2. Simulation & Path Planning

Algorithme de Balayage

Avant le vol réel, la mission a été entièrement modélisée sous Matlab pour optimiser la trajectoire.

  • Coverage Path Planning : Simulation d'un vol en "zigzag" (va-et-vient) pour couvrir 100% de la surface définie.
  • Acquisition de Données : Le script simule la capture des capteurs à intervalles réguliers sur chaque point de passage.
  • Output : Génération automatique d'une Heatmap (carte de chaleur) montrant les zones critiques (ex: zones trop sèches ou trop chaudes) sur la carte du terrain.

3. Visualisation Temps Réel

Le résultat final : un outil d'aide à la décision pour l'agriculteur.

Le Dashboard Agricole

Toutes les données (Simulation, Drone réel, Capteurs sol) sont centralisées :

  • Monitoring : Jauges d'humidité du sol en temps réel.
  • Suivi de Vol : Position GPS du drone sur la carte.
  • Alerting : Notification immédiate si l'humidité descend sous le seuil critique.
